Concepts in Git

GitHub is a web based platform widely used for version control and collaboration. And it used to store data,it allows developers to host,share and manage code repositories.

Key Features:

  • Collaboration
  • Open Source
  • Storage
  • Community Support
  • Repository
  • Code security
  • Version Control
  • Code Backup

GIT

(Global Information Tracker)

Git is a distributed version control system that helps developers track changes in their code and collaborate with others effectively.It is the backbone for platforms like Github.

Key Features

  • Version Control
  • collaboration
  • Branching and Merging
